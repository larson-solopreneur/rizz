
# Rizz

## 概要
ストリートでの声かけ記録を管理するWebアプリケーションです。

## 技術スタック
- フロントエンド: React, TypeScript, TailwindCSS
- バックエンド: Express.js
- データベース: PostgreSQL
- ORM: Drizzle

## 環境構築手順
1. 依存関係のインストール:
```bash
npm install
```

2. データベースのセットアップ:
```bash
npm run db:push
```

3. 開発サーバーの起動:
```bash
npm run dev
```

## 開発環境

アプリケーションは以下のポートで実行されます：
- Web アプリケーション: 5000番ポート

## 主な機能
- ユーザー認証（ログイン/登録）
- 声かけ記録の作成
- 統計情報の表示
- 履歴の確認

### ビルドエラー
エラー: `Type ... is not assignable to type ...`
解決策: `npm run check`を実行してタイプエラーを確認し、必要な型定義を修正してください。

### 実行時エラー
エラー: `Port 5000 is already in use`
解決策: 
1. 既存のプロセスを確認: `ps aux`
2. 該当するプロセスを終了: `kill <PID>`
